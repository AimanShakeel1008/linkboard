# project/ — the LinkBoard codebase

This folder holds the actual, evolving LinkBoard application.

**It is intentionally empty right now.** Phase 0 is about the tools (Linux &
Git); the first code lands here in **Phase 2**, when LinkBoard begins life as
an HTTP server built from raw sockets — no framework, every step logged.

Until then, the Phase 0 practice material lives in
[../concepts/phase-00-tools-of-the-trade/playground/](../concepts/phase-00-tools-of-the-trade/playground/).
